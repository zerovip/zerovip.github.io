    我的电子生活
    2019-06-24 - 3369 字

    我一向认为明文的法则是能让人的生活有条不紊的，也让人不会迷失在选择和犹豫中，是把复杂问题变简单的办法. 本文将对我的电子生活做一个梳理. 关于网络生活，希望自己遵守且容易忘记的部分形成明文的法则；非网络部分的电子生活中，对我个人常用的工具类产品的使用方法做一个整理（同时也可以看作是安利）. 两部分都将会常常更新，记录我最新做出的决定.

    网络生活法则

    根本原则：注重个人隐私.

    隐私界限

    可以暴露的内容为我真实名字的最后一个字、我的出生年代、我的性别、我的性取向、我的所学专业以及我的兴趣爱好. 除此以外的内容应非常谨慎.

    措施

    1.对微博等网络空间中曾经暴露的信息应进行大规模清查，涉及个人隐私的内容应转成“仅自己可见”.

    2.对任意两个不同的网络平台，使用的昵称、头像应当不同，且昵称的类型与风格也应尽量不同，中文、外文、符号，严肃的、活泼的都应涉及到，防止被人把不同的平台上的信息联系到一起.

    3.不同类型的信息发表在不同的平台上，每一个平台只能暴露出我的一个侧面，且控制好友的知晓程度.

    另一个原则：注意时间控制.

    网络的角色

    网络应尽量发挥它积极的作用，包括获取知识、了解世界和放松身心；同时也应在合理的限度内，以学业为重.

    措施

    1.经常制定计划，并每天进行工作小节.

    2.使用 ActionDash (Digital Wellbing 在非 Pixel 手机上的替代品) 进行屏幕时间管理.

    具体的常用网络软件使用守则

    微博

    由于审查规则不明确，已废弃.

    Twitter

    接替微博的位置，记录生活精彩瞬间，记录迸发的灵感，阅读世界上的时事，消磨时间. 锁推使用中. 每天使用应少于 1 小时.

    豆瓣

    标记书与电影，简单备份一些对质量没有要求的照片，看看世界，消磨时间. 每天使用应少于 1.5 小时.

    知乎

    匿名答题，阅读文章，看看世界，消磨时间. 目前每天使用时间过多，应当节制，培养好自己的时间线，勇于取关. 每天使用应少于 1 小时.

    QQ

    保持在线，刷刷 QQ 空间. 每天使用应少于 1 小时.

    微信

    中午晚上检查消息，阅读公众号文章. 每天使用应少于 0.5 小时.

    Telegram

    保持在线，阅读频道消息. 每天使用应少于 0.25 小时.

    bilibili

    看视频放松与学习，不固定时间.

    懂球帝

    偶尔读一下文章，不固定时间.

    网易云音乐

    想听歌了就听一下放松一下，不限时间. 学习的时候禁止听歌.. (但可以静音播放把等级刷上去.)

    Reddit

    脱离微博的那段日子靠它打发时间，现在微博戒断却因读英语比较累而逐渐放弃. 虽然少玩软件是好事，但这个理由总归不好，英语阅读应慢慢适应而不是逃避. 每天使用应多于 10 分钟.

    Facebook、Messenger、Instagram

    偶尔看看笔友的动态，看看喜欢的球队球员以及影视演员的消息. 因为朋友不多也不大玩.

    与网络关系不大的部分

    这个部分为一些与网络关系不大的电子生活的生活方式的整理，包括做哪类事情用什么软件，怎么使用那个软件，形成文字以避免功能上的交叉重叠，也方便因为意外更换电子设备时常用环境的快速搭建.

    系统

    主要使用的计算机操作系统：Manjaro

    我是一个有系统洁癖的人，不仅希望能永远首先用上最新版本的软件，同时还希望我的计算机完完全全在我的掌控之下，我希望自己知道自己的每一个操作都是在干什么. 更一般地讲，我在生活上也是一个希望清楚地知道发生了什么的人. 然而想要知道得越多，就需要额外花费越多的时间和精力，我的生活的本质就是在这个矛盾之间寻找一个平衡. 在系统的选择问题上，本来选择的是 Arch Linux，但其繁琐的驱动问题使得我退而求其次，选择了 Manjaro. 这是 DEY 学长在 2017 年 3 月推荐给我的，从此我一直在用，以后也会一直主要使用，非常感谢. 自我使用以来，自己写了非常多的各种软件的配置文件以及一些脚本，它们被我备份在 GitHub 仓库里以防出现意外后可以快速恢复出一个一模一样的环境.

    次要使用的计算机操作系统：Windows

    很多游戏不支持 Linux，还要到 Windows 上玩，还有一些例如学校任务不得不安装的软件有时候也都没有 Linux 版本.

    主要使用的手机操作系统：Android

    我是 Google 的脑残粉啦.

    工具类软件

    Linux 上常用的工具

    选择阵营其实很无聊，我 Linux 上的所有软件在这里列出. 可以看到，我是一个使用如下工具的人：zsh, vim, firefox…

    备份软件：FileGee

    FileGee 是我的 Windows 备份软件，它的免费版已经足够强大，可以支持硬盘备份、网盘备份. 我是用它配合百度网盘进行只增不减的备份的.

    输入法：小鹤双拼

    全拼感觉很慢，五笔又太难，于是学习双拼. 查的时候据说是小鹤双拼最厉害，带形还能打得更快，于是花了几天学习了一下，不熟练地用上一个月就熟练了. 我现在还没有加形.

    知晓体育比分：SofaScore

    Android 软件. 功能极其全面，有各种运动类型，各种球队、球员、比赛数据，关注一些俱乐部、球员则可以收到与他们有关的事件的提醒，极为方便.

    收藏文章稍后阅读：Instapaper

    Android 软件，但其实是全平台打通的，建一些文件夹分类，随时把文章收藏进来，随时可以看. 他们的团队对这个软件很不上心（当然了，人家很忙也很辛苦），我给他们发邮件提了两个建议（分别是长按菜单的问题与数学显示的支持），他们马上回了邮件说第一个建议下一个版本就改，第二个建议再考虑. 然而过了半年多，才发布了新版本解决了第一个很简单的问题.

    问答网站：Stack Exchange

    Android 软件，我主要在使用数学板块 Mathmatics，但实际上我的使用方法是遇到数学问题 Google 搜，就会出这个网站的结果，点进去看，喜欢的觉得有价值的回答就收藏到那个 Instapaper 里（这也是为什么我会提建议让支持数学显示，不过即便不支持，我也是可以选择“在浏览器打开”，这样就跳转到 Stack Exchange 了）. 我还没有在上面留下过痕迹，只有 1 个声誉值，貌似只能提问题，但我有点不敢在这么专业的平台上提问题，更何况大多数问题 Google 搜就能解决了.

    RSS 阅读器：Inoreader

    Android 软件，用来订阅 RSS 源. RSS 大法好，必须在这里鼓吹一波 RSS 这个上个时代的互联网技术. 我订阅的源主要有：知乎日报、Engadget、果壳精选、Mathematics-Quanta Magazine 等. 本站未来将会支持 RSS 订阅.

    画数学曲线：ArityMOD

    强大方便.

    手机查看邮件：Outlook

    我常用的 163, outlook, hotmail, gmail 都可以绑定，没有广告，页面干净，显示邮件的逻辑很合理. 发邮件以及回复邮件应在电脑的官方网页端进行.

    二次验证软件：Authy

    有一个总的帐号，不会因为换手机而全部丢失. 千万不要用 Google 的，和 Google 帐号不绑定，换手机就全没了.

    备份照片：(Google) 相册

    免费的无限空间高画质存储，高画质已经比我目前的分辨率高了，很够用.

    扫描与备份手写笔记：扫描全能王

    使用 .edu 邮箱注册相当于半个高级版，无水印，空间大，再做做它的任务，非常够用. 而且支持分类，非常方便.

    跑步：悦跑圈

    一个朋友推荐给我的，一直在用.

    突破互联网限制的技术：Project V

    我个人使用 Project V 这个工具，配合自己租的一台 VPS，自己用木头、铁钉和锤子等相关工具制作了一架梯子. 手机端使用的客户端为 BifrostV.

    连接 VPS：JuiceSSH

    强大方便.

    手机查词典：MDict

    Android 软件，可以自己把词典文件 .mdx 放进去，不需要联网，查词方便，选中一个单词之后的菜单里就有 MDict 的选项，可以呼出一个迷你的页面.

    电脑查词典：GoldenDict

    Linux 和 Windows 应该都有，也是把词典文件 .mdx 放到一个文件夹里，不需要联网，联网可获取维基百科的条目，在其他软件可以快速取词呼出，我设置的是连按两次 Ctrl + C.

    做种：qBittorrent

    Linux 和 Windows 应该都有，下载种子并做种用的，是我玩 pt 站的工具. 目前仅有北邮人一个帐号，想要 M-Team 的邀请. 真心认为去中心化（BT）才是互联网的核心精神！

    背单词：Anki

    各个平台都有，手机端叫 AnkiDroid，主要使用 Linux 平台下的.

    游戏平台：Steam

    虽然别的平台也有帐号，也领一些“喜加一”的游戏，但目前只在 Steam 上买过游戏，也只在 Steam 上玩过游戏.

    （会长期玩的）游戏

    一些剧情类的玩通关之后也就没什么好玩的了，这类游戏就和读书看电影一样，只能说读书看电影是生活的一部分，却不能说某一本书某一部电影是生活的一部分——除非总是翻阅它（《圣经》）/观看它（《新闻联播》）；然而有一类游戏是可以长期玩的，这类游戏当然就是我电子生活的一部分了.

    坎巴拉太空计划

    非常喜欢的游戏，Linux 和 Windows 都支持，是一个建造飞船飞向太空的游戏，会一直玩下去.

    侠盗猎车手

    我同样也非常喜欢，在线的部分非常庞大，而且常常更新，在没有玩腻之前会一直玩.

    Ingress

    一个朋友带我入的坑，是一个手机端的虚拟现实的游戏，佛系玩. 启蒙军才是人类的未来！
