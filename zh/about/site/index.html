    本站为个人博客站，只会发表个人的原创或翻译的文章，不转载文章，不接受投稿. 但本站绝不是一个封闭的博客站，我也绝不会封闭自己——每篇文章都会有必要的超链接，同时我也在整理一些其他的优秀网站（包括但不限于他人的博客），欢迎把自己或他人的优秀的博客推荐给我.

    本站支持中英法三语，默认语言为中文. 虽然看上去逼格很高，但由于我个人时间有限，学习任务繁重，且法语还不能自由使用，未来本站的文章将基本不会有多语种翻译，数学区的内容会以中文或英文书写，非数学区的内容会主要以中文书写.

    本站为静态博客，由 Hugo 生成，部署在 GitHub Pages 上，有关建站的技术细节请参考我的文章：建站记.
