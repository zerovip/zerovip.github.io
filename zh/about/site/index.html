    介绍

    本站为个人博客站，只会发表个人的原创或翻译的文章，不转载文章，不接受投稿. 不过我在整理一些优秀网站（包括但不限于他人的博客），欢迎把自己或他人的优秀的博客推荐给我.

    本站分为数学区和非数学区.

    数学区

    数学区分为数学笔记和数学科普.

    数学笔记区比较专业，里面都是笔记的 PDF 文件，都是我的学习存档，可供数学专业的同学们翻阅与交流.

    数学科普区就是面向普通读者的了，希望在这里能把优美的数学介绍给每一位普通读者，读者大致需要高中左右的数学知识水平，再加上一点耐心就可以读懂这里面的文章. 在这里的文章的最开头我会简要说明阅读这篇文章需要知道的数学知识，并给出几道简单的题目，如果读者对这些知识不陌生，能回答出这些题目，那么读这篇文章就是没问题的.

    非数学区

    非数学区就比较混乱，所有和数学无关的文章都塞进这里，分为日志、写作和备忘三部分.

    日志区的文章都是我记录自己点滴生活的文字，适合希望了解我的生活近况的读者阅读，如果这里出现了数学，也只可能是对数学的泛泛而谈，不会含有数学细节.

    写作区的文章会包含我创作的、或我翻译的小说、诗歌等，虽然可能创作的灵感来源于我的生活，但只要和我本人生活不直接相关，就会放在这里.

    备忘区的文章会更多更杂乱，只要是我学习过的、了解过的、经历过的，技术性的、知识性的、经验性的东西都会在这里，这里会包罗万象，而且这里是谈细节的地方，所以不适合任何人浏览，初衷是我自己的备忘，最好的到达这里文章的方法是遇到某个问题后通过搜索引擎到达这里.

    语言

    本站支持中英法三语，默认语言为中文. 虽然看上去逼格很高，但由于我个人时间有限，学习任务繁重，且法语还不能自由使用，未来本站的文章将基本不会有多语种翻译. 数学区的内容会以中文或英文书写，非数学区的内容会主要以中文书写.

    评论

    本站的文章支持评论，且各个语言板块下的评论不共享，即便是同一篇文章的不同语言. 因此在您评论时请注意使用当前页面的语言. 在这里我使用了以 GitHub issue 为工具的 utterances 作为评论系统，因此需要您登录 GitHub 帐号并授予写 issue 的权限给 utterances 机器人. 如果您不愿意授权，只需到这里建立以文章 path 为标题的 issue 并评论，或在已有的以文章 path 为标题的 issue 下评论即可.

    例如，这篇文章的网址是 https://zerovip.github.io/zh/about/site/，则这篇文章的 path 为 zh/about/site/，即去掉前面的 https://zerovip.github.io/.

    如果您没有 GitHub 账户，您可以免费注册一个；若您不愿意注册，也可以给我发邮件（zer0zero-blog[AT]yahoo[DOT]com 或 zer0zero[AT]sina[DOT]cn）进行交流.
