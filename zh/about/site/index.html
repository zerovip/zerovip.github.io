    语言

    本站支持中英法三语，默认语言为中文. 虽然看上去逼格很高，但由于我个人时间有限，学习任务繁重，且法语还不能自由使用，未来本站的文章将基本不会有多语种翻译. 数学区的内容会以中文或英文书写，非数学区的内容会主要以中文书写.

    评论

    本站的文章支持评论，且各个语言板块下的评论不共享，即便是同一篇文章的不同语言. 因此在您评论时请注意使用当前页面的语言. 在这里我使用了以 GitHub issue 为工具的 utterances 作为评论系统，因此需要您登录 GitHub 帐号并授予写 issue 的权限给 utterances 机器人. 如果您不愿意授权，只需到这里建立以文章 path 为标题的 issue 并评论，或在已有的以文章 path 为标题的 issue 下评论即可.

    例如，这篇文章的网址是 https://zerovip.github.io/zh/about/site/，则这篇文章的 path 为 zh/about/site/，即去掉前面的 https://zerovip.github.io/.

    如果您没有 GitHub 账户，您可以免费注册一个；若您不愿意注册，也可以给我发邮件（zer0zero[at]sina[dot]cn）进行交流.
