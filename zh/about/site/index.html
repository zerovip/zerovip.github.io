介绍 ¶

本站为个人博客站，只会发表个人的原创或翻译的文章，不转载文章，不接受投稿. 不过我在整理一些优秀网站（包括但不限于他人的博客），欢迎把自己或他人的优秀的博客推荐给我.

本站分为数学区和非数学区.

数学区 ¶

数学区分为数学笔记和数学科普.

数学笔记区比较专业，里面都是笔记的 PDF 文件，都是我的学习存档，可供数学专业的同学们翻阅与交流.

数学科普区就是面向普通读者的了，希望在这里能把优美的数学介绍给每一位普通读者，读者大致需要高中左右的数学知识水平，再加上一点耐心就可以读懂这里面的文章. 在这里的文章的最开头我会简要说明阅读这篇文章需要知道的数学知识，并给出几道简单的题目，如果读者对这些知识不陌生，能回答出这些题目，那么读这篇文章就是没问题的.

非数学区 ¶

非数学区就比较混乱，所有和数学无关的文章都塞进这里，分为日志、写作和备忘三部分.

日志区的文章都是我记录自己点滴生活的文字，适合希望了解我的生活近况的读者阅读，如果这里出现了数学，也只可能是对数学的泛泛而谈，不会含有数学细节.

写作区的文章会包含我创作的、或我翻译的小说、诗歌等，虽然可能创作的灵感来源于我的生活，但只要和我本人生活不直接相关，就会放在这里.

备忘区的文章会更多更杂乱，只要是我学习过的、了解过的、经历过的，技术性的、知识性的、经验性的东西都会在这里，这里会包罗万象，而且这里是谈细节的地方，所以不适合任何人浏览，初衷是我自己的备忘，最好的到达这里文章的方法是遇到某个问题后通过搜索引擎到达这里.

语言 ¶

本站支持中英法三语，默认语言为中文. 博客设计与建设时的原意是计划每篇文章都做三语翻译，使得同样的内容可以通过不同语言读到. 但建站完成投入使用后发现，我个人时间有限，学习任务繁重，且法语还不能自由使用，当初的设想过于不切实际. 所以未来很长一段时间内本站的文章将基本不会有多语种翻译. 数学区的内容会以中文或英文书写，非数学区的内容会主要以中文书写.

不过，我会保证以其他语言写成的文章即便不做中文翻译也会在中文区把题目翻译出来并给出链接，使得中文区列出的所有文章即是本站发表的所有文章，切实巩固中文作为本站默认语言的地位.

在任何页面下都可以在[左边栏的右上角选项部分]查看[该页面的各语种翻译情况]并进行跳转，其中 ⦿ 为当前页面的语言，○ 为[可以跳转的][对当前页面进行翻译了的]语言，⨂ 为[还没有对当前页面进行翻译]的语言.

评论 ¶

本站的文章支持评论，且各个语言板块下的评论不共享，即便是同一篇文章的不同语言. 因此在您评论时请注意使用当前页面的语言. 在这里我使用了以 GitHub issue 为工具的 utterances⧉ 作为评论系统，因此需要您登录 GitHub 帐号并授予写 issue 的权限给 utterances 机器人. 如果您不愿意授权，只需到这里⧉建立以文章 path 为标题的 issue 并评论，或在已有的以文章 path 为标题的 issue 下评论即可.

例如，这篇文章的网址是 https://zerovip.github.io/zh/about/site/，则这篇文章的 path 为 zh/about/site/，即去掉前面的 https://zerovip.github.io/.

如果您没有 GitHub 账户，您可以免费注册一个⧉；若您不愿意注册，也可以给我发邮件（zer0zero-blog [AT] yahoo [DOT] com 或 zer0zero [AT] sina [DOT] cn）进行交流. 如果希望与我进行加密的邮件交流，请到关于我页面查看我的公钥.

订阅 ¶

本站支持 RSS 2.0，您可以订阅任意一个列表，只需在左边栏点击想订阅的列表，进入分区后，再点击左边栏右上角的“RSS 订阅”即可.

技术 ¶

本站为静态博客，由 Hugo 生成，部署在 GitHub Pages 上. 有关建站的技术细节请参考我的文章：建站记.

本站特意把一切资源（网站样式、文章、评论、图床）都托管在 GitHub 上，且不另设域名，为的是方便统一管理，不用对不同的组件操不同的心. 为了照顾 GitHub 有时被中国大陆地区的防火墙限制的情况发生，又单独部署了一份在 Netlify 和 Vercel 上，但其中的评论与图床依旧是 GitHub 上的——所以无论如何，访问本站的体验都与访问 GitHub 的体验息息相关，而访问 Github 体验的问题还需要读者自行解决.
